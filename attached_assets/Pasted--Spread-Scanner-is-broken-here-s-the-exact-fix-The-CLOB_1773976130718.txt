**Spread Scanner fix — stop hitting CLOB, use Gamma data correctly:**

```javascript
// Fetch 500 markets not 200 — more data = more spread variety
GET https://gamma-api.polymarket.com/markets?active=true&closed=false&limit=500&order=volume24hr&ascending=false

// Then filter like this:
const spreadMarkets = markets
  .filter(m => {
    const bid = parseFloat(m.bestBid)
    const ask = parseFloat(m.bestAsk)
    const vol = parseFloat(m.volume24hr)
    const spread = ask - bid
    
    return (
      bid > 0 && ask > 0 &&        // has actual bid/ask data
      bid > 0.05 && ask < 0.95 &&  // not a dead market
      spread > 0.01 &&              // has some spread
      vol > 1000 &&                 // some real volume
      m.active === true &&
      m.closed === false
    )
  })
  .map(m => ({
    question: m.question,
    slug: m.slug,
    bestBid: parseFloat(m.bestBid),
    bestAsk: parseFloat(m.bestAsk),
    spread: parseFloat(m.bestAsk) - parseFloat(m.bestBid),
    volume24hr: parseFloat(m.volume24hr),
    url: `https://polymarket.com/event/${m.slug}`
  }))
  .sort((a, b) => b.spread - a.spread)
  .slice(0, 30) // top 30 widest spreads
```

Display spread in cents not decimals — multiply by 100. So 0.06 shows as "6¢".
