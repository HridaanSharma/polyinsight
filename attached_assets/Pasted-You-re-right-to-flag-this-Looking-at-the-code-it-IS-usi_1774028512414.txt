Also add this cache-busting header in your proxy server for the Gamma route:

```javascript
app.get('/proxy/gamma/markets', async (req, res) => {
  const params = new URLSearchParams(req.query)
  params.delete('_t') // remove cache buster before sending to Gamma
  
  const response = await fetch(
    `https://gamma-api.polymarket.com/markets?${params}`,
    {
      headers: {
        'Accept': 'application/json',
        'User-Agent': 'Mozilla/5.0',
        'Cache-Control': 'no-cache' // force Gamma to return fresh data
      }
    }
  )
  
  const data = await response.json()
  
  // Tell browser never to cache this response
  res.set('Cache-Control', 'no-store')
  res.json(data)
})
```

This guarantees every time Refresh is clicked, it hits Gamma fresh, gets live market data, and sends that live data to Claude. Claude never sees old or cached markets.
