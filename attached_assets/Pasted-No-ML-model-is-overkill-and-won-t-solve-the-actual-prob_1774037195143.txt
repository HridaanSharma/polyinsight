The real problem is simple: **there are genuinely only 4-5 cross-category causal relationships possible with today's Polymarket markets.** That's not a model problem, that's a data problem.

Here's the honest breakdown of what's actually live on Polymarket right now:

- ~40% of volume = Iran/Middle East markets
- ~25% of volume = Fed/interest rate markets  
- ~15% of volume = Bitcoin/crypto price markets
- ~10% = Sports games (useless for causal chains)
- ~10% = Everything else (elections, leadership, policy)

That "everything else" 10% is where your cross-category chains come from. And right now that 10% doesn't have enough pairs to form many valid chains.

**The actual fix is not technical — it's about what you show:**

Instead of trying to find more chains from thin data, make the feature show **why** a chain matters with real numbers.

For each valid chain, add this insight panel below the markets:

```javascript
// For each chain, calculate and display:

const generateInsight = (chain) => {
  const sideAMarkets = chain.sideA.markets
  const sideBMarkets = chain.sideB.markets
  
  // Find the most uncertain market on each side
  // (closest to 50% = most tradeable)
  const mostUncertainA = sideAMarkets.reduce((best, m) => {
    const yes = parseFloat(JSON.parse(m.outcomePrices)[0])
    const distFrom50 = Math.abs(yes - 0.5)
    const bestDist = Math.abs(parseFloat(JSON.parse(best.outcomePrices)[0]) - 0.5)
    return distFrom50 < bestDist ? m : best
  })

  const mostUncertainB = sideBMarkets.reduce((best, m) => {
    const yes = parseFloat(JSON.parse(m.outcomePrices)[0])
    const distFrom50 = Math.abs(yes - 0.5)
    const bestDist = Math.abs(parseFloat(JSON.parse(best.outcomePrices)[0]) - 0.5)
    return distFrom50 < bestDist ? m : best
  })

  const yesA = parseFloat(JSON.parse(mostUncertainA.outcomePrices)[0])
  const yesB = parseFloat(JSON.parse(mostUncertainB.outcomePrices)[0])
  
  // Calculate implied conditional probability
  // If A happens (yesA%), what should B be?
  // Right now B is at yesB% — is that consistent?
  const impliedB = yesA * yesB  // naive joint probability
  
  // Check 24h price changes for divergence
  const changeA = parseFloat(mostUncertainA.priceChange24h || 0)
  const changeB = parseFloat(mostUncertainB.priceChange24h || 0)
  
  const diverged = Math.abs(changeA) > 0.03 && Math.abs(changeB) < 0.01
  
  return {
    mostUncertainA,
    mostUncertainB,
    yesA: (yesA * 100).toFixed(1),
    yesB: (yesB * 100).toFixed(1),
    impliedJoint: (impliedB * 100).toFixed(1),
    diverged,
    changeA: (changeA * 100).toFixed(1),
    changeB: (changeB * 100).toFixed(1),
    insight: diverged 
      ? `⚠️ ${mostUncertainA.question.substring(0,40)}... moved ${changeA > 0 ? '+' : ''}${(changeA*100).toFixed(1)}% today but ${mostUncertainB.question.substring(0,40)}... hasn't repriced yet`
      : `These markets imply a ${(impliedB*100).toFixed(1)}% joint probability — if you believe both are connected, check if either side is mispriced`
  }
}
```
